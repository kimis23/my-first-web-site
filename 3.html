Javascript

javascript는 웹 페이지에 동적인 기능을 추가하는 프로그래밍 언어입니다. HTML과 CSS가 웹 페이지의 구조와 스타일을 정의하는 반면, JavaScript는 사용자와의 상호작용을 가능하게 하고, 페이지의 콘텐츠를 동적으로 변경할 수 있게 합니다. 예를 들어, 버튼 클릭 시 알림 메시지를 표시하거나, 폼 데이터를 검증하는 등의 작업을 수행할 수 있습니다. javaScript는 이벤트 기반 프로그래밍을 지원하며, DOM(Document Object Model)을 통해 HTML 요소를 조작할 수 있습니다. 이를 통해 웹 페이지의 콘텐츠를 실시간으로 업데이트하거나, 사용자 입력에 따라 동적으로 반응하는 기능을 구현할 수 있습니다
